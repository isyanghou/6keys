## 🎯 項目概述

**Six Keys Criticality** 是一個基於臨界轉換理論的神經意識狀態分析框架。通過六個核心指標（ζ₁-ζ₆）量化神經網絡的臨界性特徵，為意識研究提供理論視角和實用工具。

### 🔑 六個核心指標

| 指標 | 名稱 | 描述 |
|------|------|------|
| **ζ₁** | **FELC** | 自由能極限環 (Free Energy Limit Cycle) |
| **ζ₂** | **TEB** | 資訊-能耗效率 (Thermodynamic Efficiency Balance) |
| **ζ₃** | **RFI** | Ricci曲率臨界流 (Ricci Flow Index) |
| **ζ₄** | **ECGP** | 因果滲流 (Effective Causal Graph Percolation) |
| **ζ₅** | **PWC** | 相位拓撲環流 (Phase-Wrapped Circulation) |
| **ζ₆** | **ACI** | 神經-星膠耦合臨界 (Astrocyte-Coupling Index) |

## 💬 核心問答預覽 | Core Q&A Preview

**想快速了解六鑰臨界理論？以下是最常被問到的核心問題：**

<details>
<summary><strong>🧠 Q1: 什麼是「六鑰臨界」？為什麼是六個指標？</strong></summary>

**A:** 六鑰臨界是基於臨界轉換理論的意識分析框架。選擇六個指標是因為：
- **數學約束**：SO(7)群的6維李代數結構
- **生物對應**：對應大腦的六個核心功能模組
- **臨界性**：每個指標都捕捉不同尺度的臨界轉換特徵

六個指標協同工作，形成完整的意識狀態描述空間。
</details>

<details>
<summary><strong>⚡ Q2: 這個理論如何解釋麻醉過程中的意識消失？</strong></summary>

**A:** 麻醉過程中，六個指標會依序失效：
1. **ζ₁ (FELC)** 首先受影響 - 能量代謝改變
2. **ζ₆ (ACI)** 隨後下降 - 神經-膠質耦合減弱  
3. **ζ₄ (ECGP)** 關鍵崩潰 - 因果連接斷裂
4. 其他指標連鎖反應，系統脫離臨界狀態

這解釋了為什麼意識消失是突然的，而非漸進的。
</details>

<details>
<summary><strong>🎵 Q3: 為什麼音樂能引發強烈情緒反應？</strong></summary>

**A:** 音樂通過多個維度同時影響六鑰系統：
- **節拍** → ζ₁ (能量相位) → 身體律動感
- **和聲張力** → ζ₆ (拓撲纏繞) → 情緒高潮
- **旋律** → ζ₂ (資訊尺度) → 期待與滿足

當和聲解析時，ζ₆的拓撲纏繞瞬間釋放，產生「雞皮疙瘩」效應。
</details>

<details>
<summary><strong>🔬 Q4: 這個理論可以被實驗驗證嗎？</strong></summary>

**A:** 完全可以！理論提供了具體的可測量預測：
- **臨界閾值**：意識轉換應在特定 D_w 值發生
- **六鑰協調**：所有指標應同步變化
- **可逆性**：意識恢復應沿相反路徑

目前正在進行多中心驗證實驗，初步結果支持理論預測。
</details>

## 🚀 快速開始 | Quick Start

### 📦 安裝

```bash
# 推薦方式：pip 安裝
pip install sixkeys

# 或從源碼安裝
git clone https://github.com/isyanghou/6Keys.git
cd 6Keys && pip install -e .
```

### 🎮 基本使用

```python
import numpy as np
from sixkeys import SixKeysAnalyzer

# 創建分析器並分析數據
analyzer = SixKeysAnalyzer()
data = np.random.randn(1000, 64)  # 示例神經數據
results = analyzer.analyze(data)

# 查看六個核心指標
for key, value in results.items():
    print(f"{key}: {value:.4f}")
```
