## 🔑 理論核心要點

### 六個關鍵參數
1. **ζ₁ (FELC)**：自由能極限環 - 能量循環效率
2. **ζ₂ (TEB)**：熱力學效率平衡 - 信息-能耗比
3. **ζ₃ (RFI)**：Ricci 流積分 - 幾何曲率流
4. **ζ₄ (ECGP)**：有效因果梯度滲流 - 因果連接強度
5. **ζ₅ (PWC)**：相位拓撲環流 - 振盪同步性
6. **ζ₆ (ACI)**：星膠-神經耦合指數 - 膠質細胞調節
